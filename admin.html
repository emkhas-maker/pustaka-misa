📤 Cara Upload PDF ke Google Drive

1
Buka Google Drive lalu masuk ke folder Perpustakaan MISA (minta link ke admin utama jika belum punya akses).
2
Klik tombol + Baru → Upload file, pilih file PDF buku yang akan diupload. Tunggu hingga upload selesai (muncul notifikasi di pojok kanan bawah).
3
Setelah file muncul di Drive, klik kanan pada file → Bagikan → ubah akses menjadi "Siapa saja yang memiliki link" dan izin Viewer. Klik Selesai.
4
Klik kanan file lagi → Dapatkan link. Salin link tersebut. Link akan berbentuk:
https://drive.google.com/file/d/FILE_ID_INI/view?usp=sharing
5
Ambil bagian FILE_ID-nya saja (teks panjang di antara /d/ dan /view). Contoh: 1A2B3CxxxxxxxxxxxxxxxxxxxxXYZ
6
Buka form Tambah Buku di tab Koleksi, isi semua metadata, dan paste FILE_ID tadi ke kolom Google Drive ID. Simpan.
⚠️ Penting: Pastikan izin file sudah diubah ke "Siapa saja yang memiliki link" sebelum disimpan ke katalog. Jika tidak, siswa tidak akan bisa membuka file tersebut.

✏️ Tips Isi Metadata Buku

📌
Judul: Tulis sesuai sampul buku. Huruf kapital di awal kata utama.
📌
Pengarang: Nama penulis/penerbit. Untuk buku teks Kemendikbud bisa diisi "Kemendikbud".
📌
Kelas: Isi angka saja (1–6). Jika berlaku untuk semua kelas, bisa dikosongkan atau isi "-".
📌
Sinopsis: Tulis 2–3 kalimat deskripsi isi buku agar siswa tahu ini buku tentang apa.
URL deployment Google Apps Script. Dapatkan dari Apps Script → Deploy → Manage deployments.
Password disimpan di browser ini saja (localStorage). Minimal 6 karakter.
